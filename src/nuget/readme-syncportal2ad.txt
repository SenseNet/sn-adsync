To finalize the installation and get started with sensenet ECM SyncPortal2AD, please follow these steps:

(install guide is accessible online at: http://community.sensenet.com/docs/install-adsync-from-nuget)

1. Build your solution, make sure that there are no build errors.

2. Install sensenet ECM Sync Portal to AD using SnAdmin

   - open a command line and go to the \Admin\bin folder
   - execute the install-syncportal2ad command with the SnAdmin tool

   .\snadmin install-syncportal2ad   

For more information and support, please visit http://sensenet.com
